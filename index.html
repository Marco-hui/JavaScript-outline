JavaScript 知识概要

开始学习 →

知识全面

概括了JavaScript从基础（如字符串、数组、函数）到进阶（如ES5、ES6、Jquery）等全方面的知识点。

内容精简

每一课的知识点都摘取最重要、最有用的，简洁而精要。

课后练习

摘选了具有代表性的课外练习，让你的知识得到巩固提升！

相关拓展

除JavaScript外，还拓展了与之相关的一些知识，如MySql、php、git等

# 启动项目

# 克隆项目
git clone https://github.com/Marco-hui/JavaScript-outline.git

# 安装依赖
npm install

# 启动开发环境
npm run dev

注意

请确保你的 Node.js 版本 >= 8.6。
