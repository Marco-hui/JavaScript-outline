# 对象

# 新建对象

直接量（推荐）：var obj = {name:'小明',age:18}
构造函数：var obj = new Object();

# 读取属性值

obj.name; //==>小明
如果读取一个不存在的属性，返回undefined

# 给对象添加属性

obj.sex = '男';
obj.marry = false;
obj['weight'] = 60

# 循环遍历

for…in

对象的属性写入顺序和显示的顺序不同
