# Math
一个保存数学公式和信息的对象，一般用于执行数学任务
## 属性
- Math.PI //3.1415926
## 方法
- round(3.6) //4 四舍五入取整
- ceil(11.3) //12 向上取整
- floor(11.8) //11 向下取整
- random() //返回0-1之间的随机数（不包括1）
- max(num1, num2) //返回较大的数
- min(num1, num2) //返回较小的数
- abs(num) //绝对值
- pow(x,y) //x的y次方
- sqrt(num) //开平方根
## 相关数学知识
- 三角函数
  + sin(radian)
  + cos(radian)
  + tan(radian)
  + 角度与弧度的转换
    > 弧度=角度*Math.PI/180
- 勾股定理
  > 在直角三角形中，斜边的平方等于直角边的平方和
